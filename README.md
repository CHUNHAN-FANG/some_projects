**百度图像识别API调取**<br>
首先登录[百度智能云官网](https://cloud.baidu.com/)，进入控制台总览“产品服务-人工智能-图像识别”，在应用列表中创建新应用，生成AppID, API Key, Secret Key（调取接口创建客户端时的必要参数）；
python可实现本地图片（或网络图片）批量读取调用接口、返回识别结果。
